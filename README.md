## Objectif du projet  

Dans le cadre de ma formation avec OpenClassrooms, j’ai développé une application web de location immobilière nommée **Kasa**.

L’objectif est de concevoir une application moderne, performante et dynamique en utilisant **React**, avec une navigation fluide et des composants réutilisables.

---

## Contexte  

Je travaille en tant que développeur front-end freelance pour Kasa, une plateforme de location d’appartements entre particuliers.

Le site existant étant ancien, une refonte complète a été lancée afin de migrer vers une stack JavaScript moderne avec **React**.

Ma mission est de développer toute l’application front-end en respectant les maquettes Figma, en utilisant des données simulées (JSON).

---

## Missions réalisées  

- Création de l’application avec React  
- Développement de composants réutilisables  
- Mise en place du routing avec React Router  
- Intégration des maquettes responsives  
- Gestion des données via un fichier JSON  
- Implémentation des fonctionnalités (Gallery, Collapse…)  
- Gestion des erreurs (page 404)  

---

## Fonctionnalités

- Navigation dynamique avec React Router
- Galerie d’images avec navigation circulaire
- Composants Collapse et Cards interactifs
- Interface responsive (mobile / tablette / desktop)
- Page d’erreur 404
